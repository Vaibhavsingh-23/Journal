# Second Brain: AI-Powered Journaling Application

Second Brain is a premium, AI-powered journaling and reflection application designed to act as your personal cognitive mirror. Unlike generic AI chatbots or statistics dashboards, Second Brain focuses on emotional depth, memory rediscovery, and personal growth using a calm, editorial design language.

## Architecture

The application is built on a modern microservices architecture, separating concerns into three distinct layers:

1. **Frontend (`/frontend-v2`)**: A React 19 application built with Vite, Tailwind CSS, shadcn/ui, and Framer Motion. It uses TanStack Query for data fetching and React Router for navigation. It features route-based code splitting, global error boundaries, and a premium editorial design language.
2. **Core Backend (`/backend`)**: A Spring Boot application providing secure JWT authentication, user management, and core CRUD operations for Journal entries. It is backed by a relational database and provides endpoints for the frontend.
3. **AI Engine (`/ai-service`)**: A Python FastAPI service that orchestrates the "Second Brain". It uses a Cognitive Orchestrator, RAG pipelines, Pinecone (Vector Database), and Google Gemini (LLM) to form long-term memories, extract insights, perform semantic search, and dynamically generate reflections based on your writing.

## Key Features

- **Distraction-Free Journaling**: A beautiful, minimal text editor that auto-saves your thoughts as you type.
- **AI Reflections**: Dynamic, conversational reflections generated by the AI based on your latest entries to help you think deeper.
- **Memory Explorer**: Rediscover past experiences intelligently grouped by themes and semantic similarity (e.g., "Trip to Bangalore", "Career Growth").
- **Insights Center**: View recurring patterns, milestones, and suggestions extracted automatically over time from your journal entries.
- **Life Timeline**: A chronological aggregator that unifies your journals, memories, and insights into one seamless narrative history.
- **Semantic Search**: Ask natural language questions like *"What did I learn about patience last year?"* and get AI-generated answers with direct source citations from your journal.

## Local Setup Guide

To run the full stack locally, you need to run all three services concurrently.

### 1. Spring Boot Backend
Provides Auth and Core CRUD (runs on `http://localhost:8080`).

```bash
cd backend
# Make sure your database properties in application.yml are set
./mvnw spring-boot:run
```

### 2. Python AI Service
Provides Cognitive RAG and Insights (runs on `http://localhost:8001`).

```bash
cd ai-service
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
# Ensure your .env has GEMINI_API_KEY, PINECONE_API_KEY, MONGO_URI
uvicorn main:app --host 0.0.0.0 --port 8001 --reload
```

### 3. React Frontend
The user interface (runs on `http://localhost:5500` or port chosen by Vite).

```bash
cd frontend-v2
npm install
# Ensure you have a .env.local file with your backend URLs
npm run dev
```

## Environment Variables

**Frontend (`frontend-v2/.env.local`)**:
```env
VITE_API_URL=http://localhost:8080/api
VITE_AI_API_URL=http://localhost:8001/ai
```

**Python Service (`ai-service/.env`)**:
```env
GEMINI_API_KEY=your_gemini_key
PINECONE_API_KEY=your_pinecone_key
MONGO_URI=mongodb://localhost:27017
MONGO_DB_NAME=journaldb
```
